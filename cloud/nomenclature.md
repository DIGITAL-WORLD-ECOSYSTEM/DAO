# Nomenclatura Oficial - ASPPIBRA DAO

Este documento estabelece o léxico obrigatório para manter a consistência entre Frontend, Backend e Agentes.

| Conceito | Termo Oficial | Uso no Código |
| :--- | :--- | :--- |
| **Organização** | ASPPIBRA DAO | `author`, `branding` |
| **Sistema** | Governance System | `README`, `titles` |
| **Usuário Autenticado** | Citizen (Cidadão) | `role`, `user_type` |
| **Ativos Reais** | RWA | `module_name`, `assets` |
| **Identidade** | Digital Identity | `auth_service` |
| **Segurança** | Edge Security | `PBKDF2`, `WebCrypto` |
| **Infraestrutura** | Cloudflare Edge | `workers`, `pages`, `d1` |

## Regras de Escrita
- **D1:** Sempre referido como `DB` no código e `governance_db` no console.
- **Ambiente:** Referenciado como `Edge Runtime` ou `Cloudflare Environment`.
