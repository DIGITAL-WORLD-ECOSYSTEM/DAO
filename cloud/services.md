# Catálogo de Serviços - ASPPIBRA DAO

Lista de serviços ativos e configurados no ecossistema atual.

## 🔴 Serviços de Núcleo (Core)
- **Identity Provider (IdP):** Sistema de login nativo via WebCrypto PBKDF2.
- **Governance API:** Hono Router gerenciando propostas e auditoria.
- **Relational Storage:** Cloudflare D1 (SQLite) para dados estruturados.

## 🟠 Serviços de Ativos (RWA)
- **Asset Registry:** Registro de ativos físicos tokenizados (Módulos Agro/RWA).
- **Blob Storage:** Cloudflare R2 para documentos e metadados de ativos.

## 🟡 Serviços de Performance
- **Edge Caching:** Cloudflare KV para sessões e respostas rápidas.
- **SEO Engine:** Gerador de metadados dinâmicos para indexação Google.

## 🟢 Serviços de Governança
- **Audit Logs:** Registro forense de todas as ações administrativas.
- **Elite Agent System:** Sistema de 8 agentes para manutenção e evolução.
